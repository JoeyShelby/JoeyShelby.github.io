1 定义

散列表（Hash table，也叫哈希表），是根据关键码值(Key value)而直接进行访问的数据结构。也就是说，它通过把关键码值映射到表中一个位置来访问记录，以加快查找的速度。

2 存储结构

  1. 数组 + 链表
  2. 数组 + 树

3 实际应用

有一个公司，当有新的员工来报道时，要求将该员工的信息加入(id,性别，年龄，住址..)，当输入该员工的id时，要求查找到该员工的所有信息。
